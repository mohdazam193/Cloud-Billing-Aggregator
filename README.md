# Cloud Billing Aggregator (SRE Capstone)

A Kubernetes-native, multi-cloud cost aggregation platform that visualizes
real AWS and Azure billing data.

## Key Highlights
- Login-based UI
- Cloud credential onboarding
- Real AWS & Azure billing APIs
- Cost-over-time charts
- Service-level breakdown
- Custom date range support
- React + Tailwind + Recharts
- FastAPI backend
- Kubernetes deployment
- GitHub Actions CI/CD

⚠️ Demo-only credential handling (session memory).
